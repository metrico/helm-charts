Usage

Helm must be installed to use the charts. Please refer to Helmâ€™s documentation to get started.

Once Helm is set up properly, add the repo as follows:

helm repo add metrico https://metrico.github.io/helm-charts

You can then run helm search repo metrico to see the charts.

Chart documentation is available at qryn.dev.
